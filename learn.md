# Portable Dev Env
This file is best viewed with the slide CLI tool: `https://github.com/maaslalani/slides`

`$ slides learn.md`

| How To | Do this    |
| ------ | ------ |
| `n`   | Next slide |
| `p`   | Previous slide |
| `gg`   | Return to this slide |
| `2G`   | Go to 2nd slide |
| `/term`   | Search for `term`, `ctrl+n` for next |
